# Location Tracker

```Created with OpenCode```

Starting prompt for planning: ```i want to build a system that allows multiple users to share their current locations - each user sends their location and receives updates of the positions of other users. use websockets for real time communication```

Real-time location sharing system built with TypeScript, Socket.IO, and React.

## Features
- Real-time location sharing between multiple users
- WebSocket communication for instant updates
- Interactive map with all participant positions
- TypeScript for type safety and better development experience

## Tech Stack
- **Backend**: Node.js, TypeScript, Express, Socket.IO
- **Frontend**: React, TypeScript, Socket.IO Client, Leaflet.js
- **Real-time**: WebSocket connections

## Getting Started

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

The backend will run on port 3001 and frontend on port 3000.
